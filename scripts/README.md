# Scripts Directory

This directory contains utility scripts for site maintenance and development.

## Files

- **convert_audio.py**: Audio file converter using pydub
  - Converts FLAC files to MP3 for better browser compatibility
  - Run: `python convert_audio.py`
  - Dependencies: `pip install pydub`

## Usage

These scripts are for development/maintenance only and should not be deployed to production servers. The .htaccess file blocks access to Python files for security.
